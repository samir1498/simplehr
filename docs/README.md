# SimpleHR Documentation

This folder contains project documentation for the `simplehr` side project.

## Structure
- `project-spec.md`: consolidated product and technical specification.
- `architecture/system-overview.md`: bounded contexts, service responsibilities, and communication model.
- `roadmap/mvp-plan.md`: phased delivery plan for building the project incrementally.
- `adr/`: architecture decision records.
